title: VS Code & Cursor Extensions
---

ArkType offers extensions you can integrate by using the **default import** and naming it `arkenv`:

```typescript
import arkenv from 'arkenv';

const env = arkenv({
  PORT: "number.port", 
  NODE_ENV: "'development' | 'production' | 'test'",
});
```
